# My e-Village

A digital village portal developed for Wai Gram Panchayat, Maharashtra.

## Features
- Agriculture information
- Education support
- Infrastructure updates
- Economy section
- Contact portal
- Marathi language support

## Technologies Used
- HTML
- CSS
- JavaScript
- Python
- Django

## Pages
- Home
- About Village
- Agriculture
- Education Support
- Economy
- Infrastructure
- Contact

## Objective
To provide a centralized digital platform for villagers to access important information and services.
